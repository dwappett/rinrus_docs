# Probe

Probe ([DOI:10.1006/jmbi.1998.2400](https://doi.org/10.1006/jmbi.1998.2400)) does contact analysis by scanning van der waals surface and identifying contacts and overlaps.

``` bash
# Example usage of probe:
$HOME/git/RINRUS/bin/probe -unformated -MC -self "all" 3bwm_h_modify.pdb > 3bwm_h_modify.probe
```

## Inputs:

- PDB file

## Outputs 

- Probe file
